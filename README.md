# 🛒 AhmadStore — Laravel E-Commerce Platform

A fully functional e-commerce web application built with **Laravel 12** and **Filament PHP**. Features a modern shopping frontend and a powerful admin dashboard for managing products, categories, brands, orders, and users.

---

## 🚀 Features

- 🛍️ Product listing with categories and brands
- 🛒 Shopping cart and checkout system
- 📦 Order management
- 🏷️ Coupon / discount support
- 🔐 Multi-role authentication (admin / user)
- 🖥️ Admin dashboard powered by Filament PHP

---

## 🧰 Tech Stack

| Technology | Purpose |
|---|---|
| Laravel 12 | Backend framework |
| Filament PHP | Admin panel |
| MySQL | Database |
| Bootstrap 5 | Frontend styling |
| Vite | Asset bundling |

---

## ⚙️ Installation

```bash
git clone https://github.com/ahmadkhairi02/LaravelProject.git
cd LaravelProject
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan db:seed
php artisan serve
```
